# vLUME Scripts

Quantitative bioimaging not only relies on high-quality images but quantitative evaluation using bespoke code. Recognising this, we included a user-definable script interpreter written in the multi-paradigm language C#. These data can be easily evaluated to give the user instant quantitative feedback about the specific sub-region of their data set (Fig. 1e).
